deltaT

IPW User Command
Category - Environmental Physics and Properties: Spatial

NAME

deltaT - adjusted diurnal temperature range

SYNOPSIS

deltaT [image]

DESCRIPTION

deltaT reads a 3 band image (default: standard input) and writes the adjusted diurnal temperature range (in degrees C) to the standard output as an 8-bit IPW image. This is used in the estimate of cloudiness and solar radiation extinction based on work by Bristow and Campbell (1984) and Running, et al. (1987).

The input image has the following bands: 

	Tmax(i)     daily maximum air temperature for day i
	Tmin(i)     daily minimum air temperature for day i
	Tmin(i+1)   daily maximum air temperature for day i+1

The output image is calculated as: 

	deltaT(i) = Tmax(i) - { Tmin(i) + Tmin(i+1) } / 2

OPTIONS

none

OPERANDS

none

EXAMPLES

          mux tmax1_img tmin1_img tmin2_img  |  deltaT  > diurnal_t_range_image

FILES

none

DIAGNOSTICS

image has bands bands, 3 band images only

The input image must have exactly three bands.
