bin2text

IPW User Command
Category - Exporting IPW Image Data

NAME

bin2text - convert binary file of integers to text file

SYNOPSIS

bin2text [ -1 ] [ -2 ] [ -4 ] [file ...]

DESCRIPTION

bin2text reads binary integers from file (default: standard input) and writes either text equivalents to the standard output. If more than one file are specified, then they are read in sequence.

OPTIONS

-1
Convert 1-byte (char) integers.

-2
Convert 2-byte (short) integers.

-4
Convert 4-byte (long) integers.

Exactly one of -1, -2, or -4 must be specified.

OPERANDS

file
Input file to read binary integers from (default: standard input).

EXAMPLES

To print the first 100 quantized pixel values from an IPW image with 2-byte pixels: 

     rmhdr | bin2text -2 | head -100

Using ipwfile, the number of bytes can be determined automatically: 

     rmhdr image | bin2text -`ipwfile -my image` | head -100

FILES

None.

DIAGNOSTICS

RESTRICTIONS

bin2text is not an IPW program, but is provided with IPW for use in shell scripts.

Input values are assumed to be unsigned.
