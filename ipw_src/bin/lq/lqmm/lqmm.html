lqmm

IPW User Command
Category - Linear Quantization

NAME

lqmm - find minimum and maximum in LQ headers for set of IPW images

SYNOPSIS

lqmm image ...

DESCRIPTION

lqmm returns the minimum and maximum values of the floating point maps of all the given images. The minimum and maximum are output to the standard output.

OPTIONS

None.

OPERANDS

image ...
input image(s)

EXAMPLES

To get the min and max values of 24 hourly images of air temperature so that they can all be quantized over the same range: 

     lqmm img1 img2 img3 ... img24
