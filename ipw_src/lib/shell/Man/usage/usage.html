usage

IPW Shell-Support Command

NAME

usage - standard IPW usage message for shell scripts

SYNOPSIS

usage pgm synopsis description

DESCRIPTION

usage is a usage message generator that may be called by IPW shell scripts. The usage messages are printed in a similar format to the IPW usage message standard for C programs.

OPTIONS

None.

OPERANDS

pgm
Name of program that generated the usage message.

sysnopsis
String explaining how the program is called.

description
Description of the program.

EXAMPLES

    usage $0 'foo -n integer [ file ]' 'frobnicates a file'

FILES

None.

DIAGNOSTICS

Always exits with nonzero status.
