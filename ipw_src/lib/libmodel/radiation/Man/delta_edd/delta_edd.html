delta_edd

IPW Library Function
Library - libmodel
Category - radiation

NAME

delta_edd - delta-Eddington transformation

SYNOPSIS

#include "ipw.h"
#include "radiation.h"

void
delta_edd(
     double  *omega,   /* single-scattering albedo       */
     double  *g,       /* scattering asymmetry parameter */
     double  *tau)     /* optical depth                  */

DESCRIPTION

delta_edd computes the delta-Eddington transformations of its arguments.

RETURN VALUE

None.
