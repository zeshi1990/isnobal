ssxfr

IPW Library Function
Library - libmodel
Category - envphys

NAME

ssxfr - steady state heat transfer between 2 layers

SYNOPSIS

#include "ipw.h"
#include "envphys.h"

double
ssxfr(
     double  k1,   /* layer 1's thermal conductivity (J / (m K sec)) */
     double  k2,   /* layer 2's    "         "                       */
     double  t1,   /* layer 1's average layer temperature (K)        */
     double  t2,   /* layer 2's    "      "        "                 */
     double  d1,   /* layer 1's thickness (m)                        */
     double  d2)   /* layer 2's    "       "                         */

DESCRIPTION

ssxfr calculates the steady state heat transfer between two layers.

RETURN VALUE

heat transfer between layers (W/m^2)
