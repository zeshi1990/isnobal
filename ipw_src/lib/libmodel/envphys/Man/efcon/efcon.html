IPW Library Function
Library - libmodel
Category - envphys

NAME

efcon - calculates equivalent thermal conductivity

SYNOPSIS

#include	"ipw.h"
#include	"envphys.h"

double
efcon(
     double  k,     /* layer thermal conductivity (J/(m K sec)) */
     double  t,     /* layer temperature (K)                    */
     double  p)     /* air pressure (Pa)                        */

DESCRIPTION

efcon calculates the effective thermal conductivity for a layer accounting for both conduction and vapor diffusion. Saturation within the layer is assumed. See page 46 of reference below.

RETURN VALUE

effective thermal conductivity (J/(m K sec))
