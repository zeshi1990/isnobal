IPW Library Function
Library - libmodel
Category - envphys

NAME

sati - saturation vapor pressure over ice

SYNOPSIS

#include "ipw.h"
#include "envphys.h"

double
sati(
        double  tk)             /* air temperature (K)  */

DESCRIPTION

sati returns the saturation vapor pressure over ice (in Pa) as a function of air temperature.

RETURN VALUE

saturation vapor pressure (Pa).
