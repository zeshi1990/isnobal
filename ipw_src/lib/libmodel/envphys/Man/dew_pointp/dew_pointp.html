dew_pointp

IPW Library Function
Library - libmodel
Category - envphys

NAME

dew_pointp - dew point temperature with tolerance

SYNOPSIS

#include "ipw.h"
#include "envphys.h"

double
dew_pointp(
     double  e)      /* vapor pressure (Pa) */

DESCRIPTION

dew_pointp returns the dew point temperature for a given vapor pressure with a tolerance threshold.

RETURN VALUE

dew point temperature (K)

HISTORY

Apr 1989
Fixed to accept saturation over ice. D. Marks, USEPA, Corvallis, OR
Oct 2015
Accepts tolerance threshold. S. Havens, USDA ARS, Boise, ID
