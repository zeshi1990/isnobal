lqhwrite

IPW Library Function
Library - libipw
Category - lqh

NAME

lqhwrite - write an IPW LQH header

SYNOPSIS

#include "lqh.h"

int
lqhwrite(
	int      fd,		|* output file descriptor	 *|
	LQH_T  **lqhpp)		|* -> array of LQH pointers	 *|

DESCRIPTION

     lqhwrite writes the array of LQH headers pointed to by lqhpp

to file descriptor fd.

lqhwrite is called by IPW application programs to write LQH headers.

RETURN VALUE

OK for success, ERROR for failure
