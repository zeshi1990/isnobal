Category - mapproj

NAME

unpack_dms - convert packed DMS to decimal degrees

SYNOPSIS

#include "mapproj.h"

double
unpack_dms(
	double	dms)		|* input value in packed DMS	 *|

DESCRIPTION

unpack_dms converts the given value in packed DMS format (degrees/minutes/seconds in the format +/-DDDMMSS.SSS where DDD is the degrees, MM is minutes and SS.SSS is seconds.) to decimal degrees.

RETURN VALUE

DMS value in double precision.
