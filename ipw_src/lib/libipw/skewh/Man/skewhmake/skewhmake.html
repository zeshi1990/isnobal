NAME

skewhmake - make an IPW SKEWH header

SYNOPSIS

#include "skewh.h"

SKEWH_T *
skewhmake(
	double     angle)	|* skew angle (degrees)		 *|

DESCRIPTION

skewhmake allocates an IPW SKEW header. The header is initialized with the remaining arguments.

RETURN VALUE

pointer to new SKEW header; NULL if error
