hostorder

IPW Library Function
Library - libipw
Category - bih

NAME

hostorder - get host byte order

SYNOPSIS

#include "bih.h"

char *
hostorder(void)

DESCRIPTION

hostorder returns a string indicating the byte order (endianness) of the host system.

RETURN VALUE

"3210"
Big-endian
"0123"
Little-endian

EXAMPLES

GLOBALS ACCESSED

None.

DIAGNOSTICS

RESTRICTIONS

The return value points to static data which is overwritten by each call.
