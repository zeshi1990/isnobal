hnbands

IPW Library Function
Library - libipw
Category - bih

NAME

hnbands - number of bands in image

SYNOPSIS

#include "bih.h"

int
hnbands(
	int             fd)	|* image file descriptor	 *|

DESCRIPTION

hnbands determines the number of bands in the image currently being read/written on file descriptor fd.

hnbands may not be called before a BIH is read/written from/to fd.

RETURN VALUE

number of bands in image

GLOBALS ACCESSED
