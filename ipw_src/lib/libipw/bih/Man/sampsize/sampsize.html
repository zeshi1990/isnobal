IPW Library Function
Library - libipw
Category - bih

NAME

sampsize - compute total

SYNOPSIS

#include "bih.h"

int
sampsize(
	int		fd)	|* file descriptor		 *|

DESCRIPTION

sampsize computes the total number of bytes per sample in the image associated with the file descriptor fd

RETURN VALUE

total # bytes per image sample
