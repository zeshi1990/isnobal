IPW Library Function
Library - libipw
Category - main

NAME

usage - print usage message and exit

SYNOPSIS

#include "getargs.h"

void
usage(void)

DESCRIPTION

usage prints a usage message on the standard error output, then exits with a nonzero status. The usage message contains the command name, a one line description of the command, and a synopsis of the command's options.

RETURN VALUE

This function does not return; it terminates the program by calling the ipwexit function.
