ltoi

IPW Library Function
Library - libipw
Category - math

NAME

ltoi - convert long int to int

SYNOPSIS

#include "ipw.h"

int
ltoi(
	long            i)	/* long to be converted to int	 */

DESCRIPTION

ltoi converts its long argument to an int. If the long integer value won't fit in an int, execution terminates with an IPW error message.

ltoi is useful for converting long values in IPW image headers into ints for use in loops, subscripts, etc., without generating scads of superfluous "lint" warnings.

RETURN VALUE

The int equivalent of "i" is returned.

EXAMPLES

GLOBALS ACCESSED

None.

DIAGNOSTICS

<i> won't fit in an "int"

RESTRICTIONS

None.
