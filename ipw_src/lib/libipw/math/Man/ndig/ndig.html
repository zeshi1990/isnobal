ndig

IPW Library Function
Library - libipw
Category - math

NAME

ndig - number of digits in an integer

SYNOPSIS

#include "ipw.h"

int
ndig(
	int i)

DESCRIPTION

ndig calculates the number of decimal digits needed to represent the value of i, including the minus sign for negative values.

RETURN VALUE

number of decimal digits in i
