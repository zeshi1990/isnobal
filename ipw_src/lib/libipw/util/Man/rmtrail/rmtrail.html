IPW Library Function
Library - libipw
Category - util

NAME

rmtrail - remove trailing white space from string

SYNOPSIS

#include "ipw.h"

void
rmtrail(
	char *s)

DESCRIPTION

Rmtrail writes an EOS after the last (i.e., rightmost) non-blank character in "s".

RETURN VALUE

None.
