IPW Library Function
Library - libipw
Category - util

NAME

rmlead - skip leading white space in string

SYNOPSIS

#include "ipw.h"

char *
rmlead(
	char *s)

DESCRIPTION

rmlead skips leading white-space characters (space, tab, carriage-return, newline, formfeed) in s.

s must be EOS-terminated.

RETURN VALUE

pointer to the first non-white-space character in s

GLOBALS ACCESSED

None.
