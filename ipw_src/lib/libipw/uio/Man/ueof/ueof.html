IPW Library Function
Library - libipw
Category - uio

NAME

ueof - check for UIO end-of-file

SYNOPSIS

#include "ipw.h"

bool_t
ueof(
	int             fd)	|* file descriptor		 *|

DESCRIPTION

ueof tests whether there any any data remaining to be read from file descriptor fd, which must have been opened for reading by uropen.

ueof is typically called after an application program has finished reading an image, to ensure that the header correctly represented the size of the image.

RETURN VALUE

TRUE End-of-file has been reached

FALSE End-of-file has not been reached
