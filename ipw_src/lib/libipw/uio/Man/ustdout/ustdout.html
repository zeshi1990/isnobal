NAME

ustdout - initialize UIO on standard output

SYNOPSIS

int
ustdout(void)

DESCRIPTION

ustdout initializes the UIO package for writing the program's standard output.

ustdout should be called by an IPW main() if the program will be writing image data to the standard output. ustdout should NOT be called if the program uses UNIX stdio output functions (e.g., printf).

ustdout causes program termination if any errors are encountered.

RETURN VALUE

writable UNIX file descriptor
