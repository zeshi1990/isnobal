IPW Library Function
Library - libipw
Category - uio

NAME

mktemplate - create a template for the name of a temporary file

SYNOPSIS

char *
mktemplate(const char * prefix)		|* prefix for file name *|

DESCRIPTION

mktemplate creates a template for the name of a temporary file. The format of the template is "/tmp/{prefix}.XXXXXX". The template returned by mktemplate is then passed to the uwopen_temp() function to create the actual name for a temporary file and to open it.

The caller is responsible for deallocating the returned template.

RETURN VALUE

non-NULL
New template.

NULL
Insufficient memory to create the template.

EXAMPLES

Typical usage: 

    char * template;
    template = mktemplate("foo");
    if (template == NULL)
        error("Unable to create template for temporary file");
    int fdt;
    fdt = uwopen_temp(template);
    ...
    uclose(fdt);
    SAFE_FREE(template);

GLOBALS ACCESSED

DIAGNOSTICS

RESTRICTIONS

Uses "/tmp" as the directory.
