Category - uio

NAME

uwflush - flush buffered UIO output

SYNOPSIS

#include "ipw.h"

int
uwflush(
	int     fd)		|* file descriptor		 *|

DESCRIPTION

uwflush forces any pending UIO output to be written to file descriptor fd.

uwflush is usually called to force pending output down a pipe before beginning a potentially lengthy calculation.

RETURN VALUE

OK for success

ERROR for failure
