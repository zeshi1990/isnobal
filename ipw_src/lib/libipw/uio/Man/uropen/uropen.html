NAME

uropen - open UNIX file for reading

SYNOPSIS

#include "ipw.h"

int
uropen(
	const char      *name)	|* file name			 *|

DESCRIPTION

uropen opens the file name for reading. The name "-" is understood to mean the standard input.

Use uropen wherever you would normally use the UNIX system call open() to open a file for reading.

RETURN VALUE

>= 0 A readable UNIX file descriptor for successful open.

ERROR An error occurred.
