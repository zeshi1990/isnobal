IPW Library Function
Library - libipw
Category - uio

NAME

ucopy - copy between open files

SYNOPSIS

#include "ipw.h"

*	long
*	ucopy(
*		int	      fdi,	|* input file descriptor	 *|
*		int           fdo,	|* output file descriptor	 *|
*		REG_3 long    ncopy)	|* # bytes to [try to] copy	 *|

DESCRIPTION

ucopy copies ncopy bytes from file descriptor fdi to file descriptor fdo.

Use ucopy to transfer data unconditionally between open files.

RETURN VALUE

>= 0 number of bytes copied

ERROR an error occurred.
