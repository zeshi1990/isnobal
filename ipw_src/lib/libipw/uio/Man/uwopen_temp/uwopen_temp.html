uwopen_temp

IPW Library Function
Library - libipw
Category - uio

NAME

uwopen_temp - create a temporary UNIX file and open it for writing

SYNOPSIS

int
uwopen_temp(char * template)	|* template for file name *|

DESCRIPTION

uwopen_temp creates a temporary file by passing the given filename template to the Standard C library function mkstemp(). The template should be created by calling the mktemplate() function. mkstemp() generates a filename in the template itself, opens the file for writing, and returns the new file descriptor. uwopen_temp then initializes the file descriptor for IPW I/O.

RETURN VALUE

>= 0 A writable UNIX file descriptor for successful open.

ERROR Error occurred.

EXAMPLES

Typical usage: 

    char * template;
    template = mktemplate("foo");
    if (template == NULL)
        error("Unable to create template for temporary file");
    int fdt;
    fdt = uwopen_temp(template);
    ...
    uclose(fdt);
    SAFE_FREE(template);
