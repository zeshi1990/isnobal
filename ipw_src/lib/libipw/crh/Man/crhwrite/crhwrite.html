crhwrite

IPW Library Function
Library - libipw
Category - crh

NAME

crhwrite - write an IPW CRH header

SYNOPSIS

#include "crh.h"

int
crhwrite(
	int     fd,		|* output file descriptor	 *|
	CRH_T **crhpp)		|* -> array of CRH pointers	 *|

DESCRIPTION

     crhwrite writes the array of CRH headers pointed to by crhpp

to file descriptor fd.

crhwrite is called by IPW application programs to write CRH headers.

RETURN VALUE

OK for success, ERROR for failure

EXAMPLES

GLOBALS ACCESSED

None.

DIAGNOSTICS

RESTRICTIONS

None.
