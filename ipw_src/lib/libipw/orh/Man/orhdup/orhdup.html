Library - libipw
Category - orh

NAME

orhdup - duplicate an IPW ORH header

SYNOPSIS

#include "orh.h"

ORH_T **
orhdup(
	ORH_T         **oldhpp,	|* -> old ORH array		 *|
	int             nbands)	|* # header bands		 *|

DESCRIPTION

orhdup creates a duplicate of the IPW ORH header pointed to by "old".

orhdup is an easy way for applications programs to create a new ORH (e.g., for an output image, after having read the ORH for an input image).

RETURN VALUE

A pointer to the duplicate header is returned. NULL is returned if an error occurs.
