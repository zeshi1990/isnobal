orhfree

IPW Library Function
Library - libipw
Category - orh

NAME

orhfree - free all left over parts of a OR header

SYNOPSIS

     #include "orh.h"

     int
orhfree(
	ORH_T **orhpp,
	int     nbands)

DESCRIPTION

orhfree frees an entire orhpp structure.

RETURN VALUE

     OK for success, ERROR for failure
