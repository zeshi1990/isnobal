fpmap

IPW Library Function
Library - libipw
Category - fpio

NAME

fpmap - access fpio fpixel[pixel] map arrays

SYNOPSIS

#include "fpio.h"

fpixel_t **
fpmap(
	int             fd)  	|* image file descriptor	 *|

DESCRIPTION

fpmap accesses the per-band fpixel[pixel] conversion arrays associated with file descriptor fd.

If fpmap is called BEFORE any floating-point header I/O has occurred on fd, then a default pixel<->fpixel conversion will be established, which will be unaffected by subsequent floating-point header I/O.

IPW programs use fpmap if they wish to modify the default conversion(s) from pixels to fpixels. Such programs typically read pixels with pvread, apply the map(s) explicitly, then write fpixels with fpvwrite.

RETURN VALUE

pointer to band-indexed array of map arrays, or NULL for error.
