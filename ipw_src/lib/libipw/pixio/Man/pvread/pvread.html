IPW Library Function
Library - libipw
Category - pixio

NAME

pvread - read pixel vectors

SYNOPSIS

#include "pixio.h"

int
pvread(
	int             fd,	  |* input file descriptor	 *|
	REG_4 pixel_t  *buf,	  |* -> pixel buffer		 *|
	int             npixv)	  |* # pixel vectors to read	 *|

DESCRIPTION

pvread reads npixv pixel vectors from file descriptor fd into buf.

pvread is the lowest level of IPW input that represents pixel values in a single format (unsigned integers).

RETURN VALUE

number of pixel vectors read; else ERROR for failure
