IPW Library Function
Library - libipw
Category - error

NAME

error - print error message and exit

SYNOPSIS

void error(format, arg, ...)
char *format;
{type} arg; ...

DESCRIPTION

Error prints the error message described by the printf-style "format" and "args" on the standard error output. If usrerr() has been called, its argument is also printed. If syserr() has been called, a UNIX system error message is also printed.

RETURN VALUE

None.
