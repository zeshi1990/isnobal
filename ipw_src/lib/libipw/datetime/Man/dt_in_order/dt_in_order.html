dt_in_order

IPW Library Function
Library - libipw
Category - datetime

NAME

dt_in_order - are two date-times in chronological order?

SYNOPSIS

#include "ipw.h"

bool_t
dt_in_order(
     datetime_t  *dt1,       /* first date-time                  */
     datetime_t  *dt2)       /* second date-time                 */

DESCRIPTION

dt_in_order determines if the two given date-times are in chronological order.

RETURN VALUE

TRUE The first date-time is either before or the same as the second date-time.

FALSE The first date-time is after the second date-time.
