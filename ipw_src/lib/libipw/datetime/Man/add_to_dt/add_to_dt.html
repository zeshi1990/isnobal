IPW Library Function
Library - libipw
Category - datetime

NAME

add_to_dt - add given

SYNOPSIS

#include "ipw.h"

void
add_to_dt(
	datetime_t     *dt,	   |* date-time			 *|
	int		days,	   |* # of days to add		 *|
	int		seconds)   |* # of seconds to add	 *|

DESCRIPTION

add_to_dt adds a given number of days and seconds to a given date and time. Either or both of the number of days and seconds can be 0 or negative.

RETURN VALUE

None.
