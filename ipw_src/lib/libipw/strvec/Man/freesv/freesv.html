NAME

freesv - free an entire strvec

SYNOPSIS

#include "ipw.h"

int
freesv(
	STRVEC_T *p)

DESCRIPTION

freesv frees the entire strvec structure.

RETURN VALUE

OK is the free was successful, ERROR otherwise.
