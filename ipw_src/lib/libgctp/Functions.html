libgctp - General Cartographic Transformation Package

This IPW library is an implementation of the General Cartographic Transformation Package (GCTP). GCTP is software developed by the USGS for cartographic applications.

This library was derived from MIT's modified version of the GCTP library distributed as part of the related code with the Geographic Resources Analysis Support System (GRASS) software.
